Email Reminders
Set up Resend to send appointment reminders to patients automatically.
Get your free Resend API key:
1. Go to resend.com → Sign up free
2. Dashboard → API Keys → Create API Key
3. Paste it below. Free tier: 3,000 emails/month.
Add this to Render → Environment Variables as RESEND_API_KEY
You can skip this step and add it later from Settings.
