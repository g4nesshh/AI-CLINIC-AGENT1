💳 Razorpay Deposit

Collect a refundable deposit when patients confirm their booking. Reduces no-shows significantly.
Add RAZORPAY_KEY_ID and RAZORPAY_KEY_SECRET to Render → Environment Variables. Get keys at razorpay.com (free account).
Patients see the payment option after booking. They can pay or skip — booking is confirmed either way.

📧 Email Reminders (Resend)

✉️ Uses Resend — free, reliable, no Gmail hassle.
Step 1: Sign up free at resend.com
Step 2: Go to API Keys → Create API Key → copy it
Step 3: In Render → your service → Environment → add RESEND_API_KEY
Step 4: Optionally add RESEND_FROM_EMAIL if you have a verified domain (leave blank to use Resend's test address)
